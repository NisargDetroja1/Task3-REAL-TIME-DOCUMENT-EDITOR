# Task 3 - REAL-TIME COLLABORATIVE DOCUMENT EDITOR 📝

This project is developed for CODTECH Full Stack Development Internship Task 3.

## 📌 Project Description

A Real-Time Collaborative Document Editor developed using Node.js and Socket.IO.
This application allows multiple users to edit the same document simultaneously with live updates.

---

## 🚀 Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* Socket.IO

---

## ✨ Features

* Real-time document editing
* Live collaboration
* Multiple users support
* Instant updates
* Frontend and backend integration
* Responsive UI

## ▶️ How To Run Project

### Step 1

Install Node.js

### Step 2

Open terminal in project folder

### Step 3

Install required packages

npm install express socket.io

### Step 4

Run server

node server.js

### Step 5

Open browser

http://localhost:3000

## 💬 How To Test
